Chinese Checkers

A digital version of the classic Chinese Checkers board game with multiplayer support.

About the Game

Chinese Checkers is a digital version of the classic board game that involves strategic movement and positioning. Developed using Unity and C#, this version offers both single-player and multiplayer modes. Players can enjoy a smooth and intuitive experience with an AI opponent or challenge their friends in a competitive multiplayer setting. The game includes a dynamic and engaging user interface and optimized mechanics for an enjoyable gameplay experience.

Game Features

  • Multiplayer and single-player modes
  • AI-powered opponents with various difficulty levels
  • Customizable game settings
  • Classic Chinese Checkers gameplay with smooth controls

Board Setup and Game Initialization

The BoardModel class manages the game's board and rules for Chinese Checkers, handling game setup, player pieces, and movement logic. It defines the board dimensions and initial piece positions based on the number of players. Upon starting the game, it sets up the board with pieces in their starting positions and handles invalid/empty positions. It also includes methods to handle difficulty settings and piece placement.

Game Logic and Piece Movement

The class implements piece movement and legal move validation using methods such as PieceMove and LegalMove. It checks if a move is valid based on the piece’s position and available directions. The piece's position is updated on the board, and listeners are notified of the move. Additionally, jump moves and regular moves are handled separately with methods like FindJumpDirection and FindMoveDirection.

Player Actions and AI Moves

The game supports both human and AI players. The AI’s moves are calculated using the Minimax algorithm, determining the best possible move based on the current game state and difficulty setting. The PlaceMove method compares the AI's board state to the human player's and updates the board accordingly. The AI moves are displayed through listeners, and the game checks for winning conditions after every move.

Game Saving and Loading

The SaveGame method allows the current game state to be serialized and stored for later use. The game’s state, including the board layout, number of players, and difficulty level, is saved in a file. The class also defines a LoadGame method (currently unimplemented) to load a saved game state and restore progress. This feature enables players to pause and continue games as desired. This structure encapsulates the core mechanics of Chinese Checkers, including game setup, piece movement, AI decision-making, and game persistence, ensuring a smooth gameplay experience.
