About the Game

Shura's Gauntlet is an action-packed fighting game with smooth combat mechanics and challenging opponents. It offers single-player and local multiplayer modes, where players battle their way through intense levels using unique characters with various abilities. Developed with Unity and C#, the game offers fluid animations and strategic gameplay.

Game Features

  • Fast-paced combat
  • Diverse characters with unique abilities
  • Single-player and multiplayer modes

Main Menu

The MainMenu class controls the behavior and interactions of a menu system in a Unity game. It manages three menu panels: mainMenu, settingsMenu, and optionsMenu, and provides functionality to switch between these menus and start the game. It also manages the volume slider, custom cursor, and audio, and interacts with the camera and other UI elements.

The script also handles various gameplay and menu functionalities, such as quitting the game, retrying a level, switching to different scenes (like "Training," "FightersHub," or "ComboDisplay"), and handling mouse input events. It also manages the cursor settings, volume control via a slider, and initializes key elements when the game starts.

The SoundManager script in Unity manages the game's audio settings, allowing players to adjust the volume from the main menu. It uses a singleton pattern to ensure only one instance of the SoundManager exists throughout the game.
